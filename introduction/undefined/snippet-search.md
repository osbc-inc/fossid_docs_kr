# Snippet Search

Snippet Search는 별도의 UI를 통해 Snippet 코드에 대한 오픈소스 매치 정보를 제공하는 기능입니다.

개발 도중에도 오픈소스 매치 정보를 빠르게 확인할 수 있으며, 라이선스 컴플라이언스 문제를 조기에 예방할 수 있습니다.
