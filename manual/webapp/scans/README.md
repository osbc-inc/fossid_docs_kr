# Scans

자신에게 할당된 모든 Scan을 확인 및 관리할 수 있고, 새 Scan을 추가할 수 있습니다.

'Edit'을 통해 Project에 맵핑 등 기존 Scan을 편집할 수 있습니다.

'Archive Scan'을 통해 업로드 된 파일을 삭제하고 검증된 정보를 보존할 수 있으며, 이를 통해 메모리 공간 확보를 할 수 있습니다.
