# License 생성

권한이 있는 User는 존재하지 않는 License를 직접 생성할 수 있습니다.

\[License] -> \[Add License]를 클릭하여 생성할 수 있습니다.

빨간색으로 표시된 Full Name, Identifier는 필수 기입 항목입니다.
