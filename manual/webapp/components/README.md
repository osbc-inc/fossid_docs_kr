# Components

FossID 웹 애플리케이션은 식별에 사용되었거나 'Add Component'를 통해 Component 데이터베이스에 추가된 모든 Component에 대한 정보를 저장합니다. Component는 일반적으로 취약점, License 등에 대한 정보와 함께 특정 버전의 소프트웨어 패키지를 나타냅니다.

권한이 있는 경우, 기존 Component를 편집, 추가할 수 있습니다.
